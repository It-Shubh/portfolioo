HTML — Structure of the Web

HTML (HyperText Markup Language) is the backbone of every web page. It gives structure and meaning to content on the web using elements called tags.

📘 Common HTML Tags

  • <h1>...</h1> — Heading tag
  • <p>...</p> — Paragraph
  • <a href> — Link
  • <img src> — Image
  • <div> — Section container

💻 Example

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first web page!</p>
</body>
</html>

✨ HTML forms the base of every web technology — once you master it, styling with CSS and scripting with JavaScript becomes easy.
